srt-owl
-------

Collect difficult (rare) words from a text or a subtitle file to learn new foreign vocabulary and to
do all the looking-up beforehand.

##Example usage

$ ./srt-owl -low 160 -high 300 -lang nl twinpeaks-s1e1-nl.srt
